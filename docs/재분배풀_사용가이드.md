# 재분배 풀(공동영업 풀) 사용 가이드

## 1. 한눈에 보기

> **계약서나 결제 청구서를 보낸 지 14일이 지났는데 아직 결제(또는 계약 확정)가 안 된 고객을 모아두는 공동 풀입니다.**
> 누구든 풀에서 고객을 **픽업**해서 마무리할 수 있고, **3일 안에 수납 완료 또는 계약 완료**가 되면 그 고객이 **본인 DB로 자동 이동**합니다.

- 위치: **대시보드 상단 "재분배 풀" 버튼**
- 버튼 옆 숫자(badge) = 현재 풀에 들어있는 고객 수
- 60초마다 자동 갱신

---

## 2. 어떤 고객이 풀에 들어오나요?

다음 **3가지** 중 하나라도 해당되면 자동으로 풀에 올라옵니다.

| # | 조건 | 설명 |
|---|------|------|
| 1 | 전자계약 발송 후 14일 경과, 아직 미서명/미수납 | `발송완료 / 서명대기 / 작성완료 / 수납대기` 상태 |
| 2 | 결제선생 청구서 발송 후 14일 경과, 아직 미결제 | 청구서 state = 대기 |
| 3 | **소급(legacy)**: 위 1·2는 아니지만, 옛날에 계약서/청구서를 발송했고 아직 활성 상태인 고객 | 미동의류, 계약서발송완료(*), 수납대기 |

**풀에서 빠지는 경우(=픽업 대상 아님)**
- 계약완료 / 집행완료 / 쓰레기통 등 종결 상태
- 이미 누군가 픽업해서 임시배정 중인 고객 (단, "내 임시배정" 탭에서는 본인이 픽업한 건이 보임)

## 3. 모달 화면 구성

```
┌──────────────────────────────────────────────────────────┐
│ 재분배 풀                                                 │
├──────────────────────────────────────────────────────────┤
│ [전체 N]  [내 임시배정 M]      🔍 이름·상호명·전화번호 검색  │
├──────────────────────────────────────────────────────────┤
│ 고객명 | 상호명 | 전화 | 원담당 | 상태 | 트리거 | 경과일 | [픽업] │
│ ...                                                       │
└──────────────────────────────────────────────────────────┘
```

- **전체 탭**: 픽업 가능한 모든 고객
- **내 임시배정 탭**: 내가 픽업해서 현재 임시 보유 중인 고객 (D-3 카운트다운 포함)
- **검색창**: 이름·상호명·전화번호 어느 것으로든 부분 검색. 공백·하이픈 자동 무시.

## 4. 사용 흐름 (스텝 바이 스텝)

### ① 풀에서 고객 픽업하기
1. 대시보드 상단 **"재분배 풀"** 클릭
2. 원하는 고객의 행에서 **[픽업]** 버튼 클릭
3. 픽업 즉시 **3일 임시배정**이 시작되고, "내 임시배정" 탭으로 이동
4. 픽업 정보(픽업자/시각/만료시각)는 자동 기록되며, 원래 담당자는 그대로 표시됨

> 한 고객은 **선착순 1명만 픽업** 가능합니다.

### ② 픽업 후 영업/마무리
1. 고객 카드를 열어 평소처럼 상담·자료요청·재발송 진행
2. **수납 완료** 또는 **계약 완료(후불/외주)** 처리 시:
   - **3일 안**이면 → 담당자가 **자동으로 본인(=픽업자)으로 변경**
   - 메모란에 "재분배 풀에서 확정 이동" 자동 기록
   - 정산은 **새 담당자 100% 적용**

### ③ 임시배정 해제하기 (필요 시)
- "내 임시배정" 탭에서 [해제] 버튼 → 원담당자에게 돌아감
- **3일이 지나면 자동으로 해제**되어 풀에 다시 올라옴

## 5. 자주 묻는 질문(FAQ)

**Q. 픽업했는데 3일 안에 못 끝내면 어떻게 되나요?**
A. 임시배정이 풀리고 고객은 원담당자에게 그대로 남습니다. 다시 픽업 가능합니다.

**Q. 픽업하면 원담당자의 정산이 어떻게 되나요?**
A. **확정 이동(수납완료/계약완료)이 일어난 시점부터** 새 담당자(픽업자) 100% 정산. 그 전까지는 변동 없음.

**Q. 원담당자가 휴가/퇴사라서 픽업했는데, 본인이 다시 영업하고 싶다면?**
A. "내 임시배정" 탭에서 [해제] 후 원담당자에게 직접 인계 가능. 또는 그대로 마무리하면 본인 건이 됩니다.

**Q. 검색은 이름만 되나요?**
A. 이름, 상호명, 전화번호 모두 부분 검색 가능합니다. 공백과 하이픈은 자동 무시됩니다. (예: "010-1234" / "01012 34" 모두 동일하게 검색)

**Q. 풀에 안 보이는데 분명 14일 지났어요.**
A. 다음을 확인해주세요:
- 상태가 종결(계약완료/집행완료/쓰레기통)인지
- 이미 누군가 픽업해서 임시배정 중인지 ("내 임시배정" 탭 또는 본인이 픽업했다면 거기 표시됨)
- 계약서/청구서 발송 이력이 실제로 있는지 (소급 후보는 발송 이력이 있어야 풀에 올라옵니다)

**Q. 한 사람이 여러 고객을 픽업해도 되나요?**
A. 됩니다. 단 한 고객당 1명만 픽업 가능합니다.

## 7. 운영 팁

- **매일 아침 풀 한번 둘러보기**: 14일 묵힌 건이 의외로 빨리 풀리는 경우가 많습니다.
- **픽업 전에 검색으로 중복 확인**: 본인이 예전에 상담했던 고객이 풀에 나와있을 수 있습니다.
- **D-3 카운트다운 관리**: "내 임시배정" 탭에서 만료일을 확인하고 우선순위 조정.
- **무리한 픽업 금지**: 3일 안에 처리 가능한 수만 픽업 권장.
