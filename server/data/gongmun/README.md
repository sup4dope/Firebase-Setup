# 정책자금 공문 디렉토리

이 폴더에 정책자금 공문을 `.txt` 또는 `.md` 파일로 저장하면, AI 채팅의 system prompt에 자동으로 첨부됩니다.

## 사용법

1. 공문 PDF의 본문을 텍스트로 추출 (복사·붙여넣기 또는 OCR)
2. 파일명은 공문 식별이 쉽도록 작성: 예) `2026_소상공인_정책자금_공고.txt`
3. 한 파일당 1개 공문 권장 (너무 길면 모델 컨텍스트 한도 초과 가능)
4. 저장하면 60초 내 자동 반영 (캐시 TTL)

## 주의사항

- `README.md`로 시작하는 파일은 무시됩니다.
- 파일이 많거나 공문이 길면 토큰이 폭증해 응답이 느려질 수 있습니다.
  → 공문이 5건 이상 누적되면 RAG(벡터 검색) 방식으로 전환을 검토하세요.
- 공문이 변경되면 파일을 직접 교체하면 됩니다 (배포 후 자동 반영).

## 향후 개선 (옵션)

- 관리자 페이지에서 업로드/삭제 UI
- PDF → 텍스트 자동 변환
- 임베딩 + 유사도 검색 기반 부분 첨부 (RAG)
